If the two sites were in perfect linkage equilibrium, the probability of getting seeing the haplotype '11' would be (0.01 * 0.01 = 0.0001).

Therefore even if the gene was on an autosome, we have only sampled 20 haplotypes, the expected value in this case assuming HWE is $20 \times 0.0001 = 0.002$. The expected number of individuals which do $\it{not}$ have haplotype '11' is $20\times0.9999 = 19.998$. Using a chi squared test, with one degree of freedom...

In [97]:
chi2.sf((np.square(-0.002) / 0.002) + (np.square(20 - 19.998) / np.square(19.998)), 1)
Out[97]:
0.9643293191355615

The probability of seeing this event assuming Linkage Equilibrium is 96.4% which is not surprising at all. Therefore we cannot reject the null hypothesis that the sites are in linkage equilibrium.

In the $1000000$ individual case, the expected number of individuals who would have '11' is $1,000,000 \times 0.002 = 2000$. The expected number of individuals who would not have '11' is $1000000 - 2000 = 998000$

chi2.sf((np.square(-2000) / 2000) + (np.square(1000000 - 998000) / 998000), 1)
Out[108]:
0.0

The probability of this event is extremely unlikely, therefore we can reject the null hypothesis that the sites are in linkage equilibrium. Thus we would have evidence that the sites are in LD.

The power of the experiment to assess the LD status of 2 sites increases as the expected number of individuals with both minor alleles increases. If the minor allele frequency is too small, then the expected number of individuals who have both minor alleles assuming HWE will be extremely small, thus decreasing the likelihood of seeing an individual with both minor alleles.

Question 4¶

For any lineage in $T_k$, it is guaranteed to have at least 1 descendent at $T_n$

Therefore there are (n-k) "free descendents" which can be assigned to any particular lineage, with any configuration of these free descendents being equally likely.

Therefore, for a random lineage to have i descendents, it must have i-1 "free descendents"

So first, we must calculate the number of ways these free descendents can be assigned to all lineage such that a particular lineage has i-1 descendents.

To do this, first subtract the i-1 free descendents from n-k.

Now there are n-k-i+1 free descendents to spread amongst the other k-1 lineages

Therefore there are C(n-k-i+1+k-2, k-2)=C(n-i-1, k-2) ways to do this

To calculate the total number of configurations, we just have to spread the n-k "free descendents" across k lineages.

Therefore there are C(n-k+k-1, k-1) = C(n-1, k-1) total configurations.

So $p_{ki} = \frac{C(n-i-1, k-2)}{C(n-1, k-1)}$

$= \frac{(n-i-1)!}{(k-2)!(n-i-k+1)!}\frac{(k-1)!(n-k)!}{(n-1)!}$

$= \frac{(n-i-1)!}{(n-i-k+1)!}\frac{(n-k)!}{(n-1)!}\frac{(k-1)!}{(k-2)!}$

$= \frac{(n-i-1)!}{(n-i-k+1)!}\frac{(n-k)!}{(n-1)!}(k-1)$

Since $i! = (i-1)!i$

$= \frac{i!}{(i-1)!i}\frac{(n-i-1)!}{(n-i-k+1)!}\frac{(n-k)!}{(n-1)!}(k-1)$

$= \frac{i!}{(i-1)!i}\frac{(n-i-1)!}{(n-i-k+1)!}\frac{(n-k)!}{(n-1)!}(k-1)$

$= \frac{(n-k)!}{(n-i-k+1)!(i-1)!}\frac{(n-i-1)!i!}{(n-1)!}\frac{k-1}{i}$

$= \frac{C(n-k, i-1)}{C(n-1, i)}\frac{k-1}{i}$

The number of mutations with exactly i ones is

$m_i = \sum_{k=2}^n \sum_{j=1}^k x_{j,k} y_{i, j, k}$

where $x_{j, k}$ represents the number of mutations that fall on the jth lineage in epoch $T_k$

And $y_{i,j,k}$ is an indicator variable that is 1 when the jth lineage has i descendents in epoch $T_k$

By linearity of expectation

$E[m_i] = \sum_{k=2}^n \sum_{j=1}^k E[x_{j,k} y_{i, j, k}]$

$E[m_i] = \sum_{k=2}^n k E[x_{j,k} y_{i, j, k}]$

Since $x_{j,k}$ is independent to $y_{i, j, k}$

$E[m_i] = \sum_{k=2}^n k E[x_{j,k}]E[y_{i, j, k}]$

$ = \sum_{k=2}^n k \frac{C(n-k, i-1)}{C(n-1, i)}\frac{k-1}{i} E[x_{j,k}]$

$ = \sum_{k=2}^n k \frac{C(n-k, i-1)}{C(n-1, i)}\frac{k-1}{i} \frac{2N}{C(k, 2)}$

$ = \frac{1}{i}\sum_{k=2}^n \frac{C(n-k, i-1)}{C(n-1, i)}\frac{2}{1}\frac{(k-1)k}{ 2} \frac{2N}{C(k, 2)}$

$ = \frac{1}{i}\sum_{k=2}^n \frac{C(n-k, i-1)}{C(n-1, i)}\frac{2}{1}\frac{(k-1)k}{ 2} \frac{2N}{C(k, 2)}$

$ = \frac{1}{i}\sum_{k=2}^n \frac{C(n-k, i-1)}{C(n-1, i)}\frac{2}{1}\frac{C(k, 2)}{1} \frac{2N \mu l}{C(k, 2)}$

$ = \frac{4N \mu l}{i C(n-1, i)}\sum_{k=2}^n C(n-k, i-1)$

$ = \frac{4N \mu l}{i C(n-1, i)}\sum_{k=2}^n C(n-k, n-k-i+1)$

$ = \frac{4N \mu l}{i C(n-1, i)}[C(0, 0-i+1) + C(1, 1-i+1) + ... + C(n-2, n-i-1)]$

$i \geq 1$ because each lineage must have at least 1 descendent since we are simulating backwards

Since $-i \lt 0$, $C(0, 0-i) = 0$

$ = \frac{4N \mu l}{i C(n-1, i)}[C(0, 0-i) + C(0, 0-i+1) + C(1, 1-i+1) + ... + C(n-2, n-i-1)]$

Since $C(n, r) + C(n, r+1) = C(n+1, r+1)$

$ = \frac{4N \mu l}{i C(n-1, i)}[C(1, 0-i+1) + C(1, 1-i+1) + C(2, 2-i+1) + ... + C(n-2, n-i-1)]$

Repeating this simplificication ...

$ = \frac{4N \mu l}{i C(n-1, i)}[C(n-2, n-1-i-1) + C(n-2, n-i-1)]$

$ = \frac{4N \mu l}{i C(n-1, i)}[C(n-1, n-i-1)]$

$ = \frac{4N \mu l}{i} \frac{C(n-1, i)}{C(n-1, i)}$

$ = \frac{4N \mu l}{i}$
